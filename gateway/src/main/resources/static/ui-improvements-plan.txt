   - File: css/cockpit.css

5. Token Counter in Footer (api.js, app.js, index.html)
   - Added logs.metrics() to api.js (calls GET /api/logs/llm-interactions/metrics)
   - Added refreshTokenCounter() in app.js that polls every 30s
   - Updated footer in index.html to show: coin-emoji {tokens} tokens | {calls} calls | avg {ms}ms

   - Files: js/api.js, js/app.js, index.html

6. Objectives — Inline Status Actions (objectives.js)
   - After renderTable(), injects action buttons into each row's last cell
   - Three buttons per row: checkmark (Complete/ACHIEVED), warning (At Risk), x (Delete)
   - Each button calls the appropriate API and re-renders
   - Date picker inputs for startDate/endDate already present in add form
   - File: js/views/objectives.js

7. Resources — Busy/Free Indicator (resources.js)
   - After renderTable(), post-processes availability cells
   - Appends colored pills: FREE (green, .pill.good) if availability >= 0.5
   - BUSY (red, .pill.bad) if availability < 0.5
   - File: js/views/resources.js
